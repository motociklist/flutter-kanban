# Demo Flutter App

Простое Flutter-приложение с одним экраном и счётчиком.

Как запустить:

1. Убедитесь, что установлен Flutter SDK: https://flutter.dev
2. В корне проекта выполните:

```bash
flutter pub get
flutter run
flutter run -d chrome
```

Файлы:
- `lib/main.dart` — исходный код приложения
- `pubspec.yaml` — зависимости

Если у вас нет устройства, используйте `flutter run -d chrome` (для web) или `flutter emulators` + `flutter run` для эмулятора.
